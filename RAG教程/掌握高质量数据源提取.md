### 提示词编写
#### **如何编写提示词**
- **角色与任务定调**
> [!NOTE] **范本**：
>你是一个专业的数据集构建专家。请阅读提供的文本（`<text>` 标签内），根据该文本生成 [X] 个高质量的问答对。

- **生成粒度与质量基准**

> [!NOTE] 范本
> 问题必须跨越段落，考察对核心逻辑、因果关系或宏观趋势的理解。拒绝提问单一数据点。
-  **绝对红线**

> [!NOTE] 范本
> 1. 问题必须能独立成句，绝对禁止出现“本文档”、“这一段”、“上述内容”、“图表中”等代词或元引用。
> 2. 答案必须直接给出客观结论，禁止重复问题，禁止出现“根据文本可知”、“作者认为”等前缀废话。
> 3. （如需 Context）上下文提取必须绝对忠于原文，连标点符号都不允许更改。


- **异常拦截**

> [!NOTE] 范本Title
> 如果输入文本是目录、参考文献、无法阅读的乱码，或者完全无法支撑构建有意义的问答对，请直接且仅返回空列表 `[]`，不要输出任何解释。


- **结构化输出契约**

> [!NOTE]  范文
> 请严格按照以下 JSON 格式输出，不要包含任何 Markdown 代码块包裹（如 ```json ），不要输出任何其他文本： [ {"question": "<独立的问题>", "answer": "<具体的答案>"} ]

``` python
from langchain_community.chat_models import ChatOpenAI  
from langchain_core.messages import SystemMessage,HumanMessage  
import time  
import random  
def llm_build(model_name='qwen-long',temperature=0.85,max_retries=0):  
    llm=ChatOpenAI(model_name=model_name,temperature=temperature,max_retries=max_retries,api_key=os.environ['API_KEY'],base_url=os.environ['BASE_URL'])  
    return llm  
def build_prompt(prompt_templ,chunk):  
    prompt=prompt_templ.replace('{{document}}',chunk).strip()  
    return prompt  
def chat(llm,prompt,debug=False,max_retries=3):  
    def do_chat(prompt):  
        response=llm.invoke([SystemMessage("你是一个有用的助手"),HumanMessage(content=prompt)])  
        return response.content  
    while max_retries>0:  
        try:  
            return do_chat(prompt)  
        except Exception as e:  
            max_retries-=1  
            sleep_seconds=random.randint(1,4)  
            if debug:  
                print(f'Chat error: {e},remain retry:{max_retries} retry after {sleep_seconds} seconds')  
            time.sleep(sleep_seconds)  
    return None
```

### 批量处理llm针对文本块生成问答对
``` python
import threading  
import concurrent.futures  
from tqdm import tqdm  
import json  
def build_qa(documents,prompt_templ,qa_ckpt_filename):  
    qa_ckpt={}  
    if os.path.exists(qa_ckpt_filename):  
        qa_ckpt=open(qa_ckpt_filename,encoding='utf-8').readlines()  
        qa_ckpt=[json.loads(line.strip()) for line in qa_ckpt if line.strip()!='']  
        qa_ckpt={item['uuid']:item for item in qa_ckpt}  
        print(f'Load {len(qa_ckpt)} qa from {qa_ckpt_filename}')  
    file_lock=threading.Lock()  
    max_workers=4  
    with concurrent.futures.ThreadPoolExecutor(max_workers=max_workers) as executor:  
        futures={doc.metadata['uuid']:executor.submit(chat,llm_build(),build_prompt(prompt_templ,doc.page_content)) for doc in documents if len(doc.page_content.replace('\n',''))>=150 and doc.metadata['uuid'] not in qa_ckpt}  
        for uuid in tqdm(futures):  
            future=futures[uuid]  
            result=future.result()  
            if result is None:  
                continue  
            item={'uuid':uuid,'raw_resp':result}  
            qa_ckpt[uuid]=item  
            file_lock.acquire()  
            try:  
                with open(qa_ckpt_filename,'a',encoding='utf-8') as f:  
                    f.write(json.dumps(item,ensure_ascii=False)+'\n')  
            except Exception as e:  
                print(f'Write qa error: {e}')  
            finally:  
                file_lock.release()  
    return qa_ckpt
```
